Punto 1 ¿Qué significa que los estilos se apliquen en cascada? ¿Cómo se aplica el concepto de herencia?

La cascada en Css es el mecanismo que controla el resultado final cuando, por existir declaraciones múltiples, surgen conflictivos sobre qué estilos se aplican a un elemento. Dicho de otra manera: si hay dos o más declaraciones que afectan a un elemento, la cascada determina cuál de ellas tiene preferencia, mayor peso, y se aplicará. La herencia es distinta de la cascada ya que involucra al árbol (DOM) del documento. La herencia en CSS es el mecanismo mediante el cual ciertas propiedades se transmiten de un elemento padre a sus hijos.

Punto 2¿Por qué es necesario utilizar un CSS de Reset?

Todos los navegadores agregan estilos a nuestro sitios. Dichos estilos, no son siempre los mismos sino que dependen del navegador que utilicemos. A causa de esto, nuestros propios estilos pueden verse afectados y provocar incosistencia en cuanto la vizualizacion del contenido. Por ello es necesario que se agreguen un reset (que esos estilos sean eliminados).

Punto 3¿Qué es el CSS box model?

El modelo de cajas es esencialmente un esquema donde cada box envuelve uno o mas elementos HTML. Cada caja tiene los siguientes elementos:

  • Margin:Área por fuera del Border
  • Border:El Border va alrededor del Padding y del Content
  • Padding:Área alrededor del Content
  • Content:Contenido del box, donde imágenes y textos aparecen

Punto 4¿Cuál es el código que hay que insertar en una hoja de estilo para poder usar WebFonts?

Cuando los diseñadores desean utilizar una tipografía que no es estandar, deben recurrir al uso de Web Fonts. Para ello, se necesita contar con la fuente y agregar en la hoja de estilos la siguiente definicion:

@font-face{ font-family: miFuente; src: url(rutaDeMiFuente); }

Punto 5¿Qué son y para qué sirven los pseudoElementos?

Los pseudo-Elements son usados para darle estilos a una parte específica de un elemeneto. Por ejemplo, si queremos darle estilos a la primer letra o línea de un elemento así como también si queremos insertar contenido antes o después de un elemento.

Punto 7¿Qué es el view port? ¿Cómo se configura? ¿qué problema soluciona?

La viewport es la parte visible de la página web. Esta varia dependiendo del dispositivo por el cual se acceda a la misma, ya que no es lo mismo acceder a un sitio por una tablet o con un celular que con la computadora. Las resoluciones de pantallas son diferentes y nuestro sitio se debiera ajustar a la misma. Debido a que actualmente y desde hace un tiempo el uso de dispositivos móviles de diversos tamaños es más frecuente, surgió el problema de hacer que un mismo sitio se vea bien en todos ellos. La introducción de HTML5 de una nueva etiqueta, hizo que de alguna manera todos los navegadores escalen el sitio a mostrar. Para establecer la etiquete viewport, la misma debe ser declarada en el head de la siguiente manera:

meta name="viewport" content="width=device-width, initial-scale=1.0"

Donde meta viewport le indica al nevagador como se debiera escalar según la dimensión de la pantalla. Y width=device-width le indica el ancho del dispositivo.

Punto 8 ¿Qué son las media Querys? Enumere los distintos tipos de medios y las principales características de cada uno de ellos.

Media Query es una técnica introducida por CSS3. Se emplea la relgla @media para incluir un bloque de código CSS si cierta condición es verdadera.

@media only screen and (max-width: 500px) { body { background-color: lightblue; } }

En este caso, solamente se mostrará un background color si el ancho máximo es de 500px. Características principales:

  • Aspect-ratio:Mira las dimensiones relativas del dispositivo expresadas como una relación de aspecto: 16:9 por ejemplo.
  • Width y height:Mira las dimensiones del área de visualización. Además pueden ser expresadas en valores mínimos y máximos.
  • Orientation:Mira si el layout es panorámico ( el ancho es mayor que el alto ) o vertical ( el alto es mayor que el ancho ). Esto nos permite ajustar los diseños para dispositivos con propiedades de giro de la pantalla como el iPhone, y otros smartphones y los tablets.
  • Resolution:Mira la densidad de los pixeles en el dispositivo de salida. Esto es especialmente útil cuando queremos aprovecharnos de las ventajas de los dispositivos que tiene una resolución mayor a 72 dpi.
  • Color, Color-index y monochrome:Encuentran el número de color o bits por color. Esto nos permite crear diseños específicos para dispositivos monocromáticos.

Punto 9En qué circunstancias se pueden utilizar las variables css. ¿Qué problemas pueden traer aparejadas? ¿Cuándo consideras que sería bueno utilizarlas?

Las variables CSS, técnicamente llamadas "propiedades CSS no estándar",permiten cambiar dinámicamente los estilos aplicados en una página y mejorar las características de las propiedades CSS estándar. Permiten asignar cualquier valor a una propiedad cuyo nombre podemos elegir libremente. Permiten reutilizar esos valores en cualquier otra propiedad gracias a la función var(). Por ejemplo:

:root { --color-principal: #06c; } #foo h1 { color: var(--color-principal); }

La función var() obtiene el valor de la propiedad cuyo nombre se indica y lo inserta en el lugar donde se llama a la función. De esta manera podemos personalizar contenido propio.
